# Chocolate-Business-Analysys
Анализ данных по продажам шоколада от Junior аналитика использованием инструментов SQL,Python,PowerBI.

# Data Analysis Project

## О проекте
Анализ данных с использованием SQL и Python

## Инструменты
- SQL
- Python (pandas)
- PowerBI (визуализация)

## Что сделано
В данном проекте я проанализировал данные,
связанные с продажей шоколада,
с целью выявления ключевых закономерностей в поведении покупателей и структуре продаж,
а так же сделал визуализацию чтобы вам было понятнее мои ответы на задачи.

## Выводы
В рамках анализа были решены следующие задачи:

определение наиболее популярных видов и брендов шоколада
анализ стран с наибольшими продажами
выявление наиболее удобных форматов магазинов для покупателей
анализ предпочтительных способов оплаты
расчет среднего количества продаж за месяц
определение общей выручки по странам

По результатам анализа были выявлены основные тенденции: различия в предпочтениях покупателей в зависимости от страны, наиболее популярные категории товаров и часто используемые способы оплаты.

Также были рассчитаны средние и суммарные показатели продаж, что позволяет оценить объемы выручки и лучше понять поведение клиентов.
